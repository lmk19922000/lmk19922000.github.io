Instructions

Dear participant , please read the instructions before the experiment.

The purpose of this experiment is to compare 2 copy paste techniques: TRADITIONAL and AUTOCOMPASTE. In the experiment, you will have to complete many copy paste tasks using both techniques.

Below is how the interface looks like:

The screen is split into two sections, the left section contains the conditions of the experiment while the right section contains the text editor window and the window containing text.

Left section

There is a red box containing words. You have to enter those words into the Text Editor window using one of the two techniques.

Right section

There are two kinds of windows in the right section: 1) Text Editor window and 2) Articles window. The Text Editor window is meant for you to enter your response using either one of the proposed copy-pasting techniques. The Articles windows contain text in various forms (point-form / essay). The text in the red box will be highlighted in yellow for easy identification in the cases of the TRADITIONAL copy-pasting technique.

TRADITIONAL technique

When using the TRADITIONAL techinique, you need to select the desired text you want in the Articles window (NOT in the red box of left section), press combination Ctrl-C. Then you move on to the Text Editor window and press combination Ctrl-V. The text will be copied from the Articles window to the Text Editor window.

AUTOCOMPASTE technique

When the AUTOCOMPASTE technique is active, typing three characters or more in the Text Editor will show a dropdown list of possible sentences that contain the characters/sentence that you have just entered, which are extracted from the contents of the Articles window. Pressing the ↑/↓ buttons on the keyboard will scroll through the list of suggestions. Press enter to select a suggestion and the text editor will display the full selected sentence.

After selecting a sentence, pressing ←/→ button right after pressing enter, you can delete/append the next few sentences of the article to the Text Editor. Pressing enter or space would mean pasting the final text to the system.

After you are satisfied with the text entered in the text editor, pressing the Next Task button will load the next tasks with its respective conditions.

In the following section, you will practice with both techniques before the actual experiment. Press Next button once you are ready.
